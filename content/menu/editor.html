Editor for 530 cache

Overview

A collection of various incomplete open-source cache editor projects, finally unified into a single, powerful editor. (Currently under development)

Features

  • Item Editor
  • NPC Editor
  • Object Editor
  • Interface Editor (Under development)
  • Transfer regions between any caches
  • Transfer Interface
  • Transfer Index
  • Export model (Under development)
  • Export list of items, npcs, objects
  • Export Indices (317-530)
  • Pack models
  • Pick a Color
  • File Manager (Under development)
  • Model Viewer

Note

This project is not intended for commercial use but rather serves as a learning experience. The code is developed for educational purposes and to demonstrate techniques that can be applied to development challenges.
