Cache Editor

Overview

A powerful new tool for managing the cache, written in Kotlin.

Available tools:

  • Item editor
  • NPC editor
  • Object editor
  • File manager

Status

This is an alpha release. Some features are incomplete or experimental.
Currently, only a manual download is available.
Suggestions, feedback, and bug reports are welcome on GitLab Issues.

Download

2.0.1-alpha+internal

Note

This project is not intended for commercial use. It is developed primarily for learning purposes and to explore and demonstrate techniques that can be applied to RSPS development challenges.
